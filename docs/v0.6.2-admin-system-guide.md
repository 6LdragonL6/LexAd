# LexAd v0.6.2 管理员系统管理指南

## AI 配置

1. 使用管理员账户登录。
2. 在侧栏打开“系统管理”，进入“AI 配置”。
3. 输入 DeepSeek API Key，点击“验证并保存”。
4. 系统使用固定的 `deepseek-v4-flash` 模型发送最小验证请求；验证失败不会覆盖当前配置。

页面只显示密钥掩码。已保存的明文 Key 无法从系统读取。如需更换，直接输入新 Key；如需停用，清除管理员配置。

若服务器仍设置了兼容旧部署的 `DEEPSEEK_API_KEY`，清除管理员配置后会回退到服务器环境变量。页面会明确显示配置来源。

生产环境必须为 `SECRET_KEY` 设置足够随机且长期稳定的值。变更 `SECRET_KEY` 会导致原加密配置无法解密，因此变更前应先清除配置，变更后重新保存。

## 删除与回收站

管理员可在工作台删除物料、在品牌档案删除品牌、在资料中心删除舆情案例或平台规则。删除后对象进入回收站：

- 保留 15 天；
- 普通业务页面、统计和新审查不再使用这些对象；
- 可以在“系统管理 → 回收站”恢复；
- 可以二次确认后立即永久删除。

永久删除不可恢复。物料永久删除会连同审核和提交快照一起清理；品牌永久删除只解除历史物料关联，不删除物料。

导入记录和审计日志不可删除。系统会记录删除、恢复、手动清理和到期自动清理动作，但不会记录完整物料或 API Key。
