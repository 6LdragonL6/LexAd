======================================================================
小红书聚光平台 - 广告审核规则与行业准入（2025版）
======================================================================

【说明】本文汇总了聚光平台广告开户资质、行业准入分类、禁投行业
及风险积分管理规则。具体执行以平台官方最新公告为准。

======================================================================
一、基本开户资质要求
======================================================================

| 资质项             | 要求                                         |
|--------------------|----------------------------------------------|
| 营业执照           | 三证合一、有效期内、注册满1个月以上          |
|                    | 经营范围与广告内容匹配                       |
| 法人身份证         | 正反面清晰扫描件                             |
| 对公银行账户       | 开户许可证或银行账户证明                     |
| 企业专业号认证     | 600元/年（蓝V标识），个体工商户暂未开放      |

开户路径：企业专业号后台 -> 营销推广 -> 聚光平台
首次充值门槛：官方直客 1万~5万元；代理商可低至 5000元
审核周期：资质齐全情况下 3~5个工作日
连续失败3次可能冻结申请

二、行业准入分类
======================================================================

[无门槛行业] 无需额外提交资质

  类别                   具体内容
  ─────────────────────────────────────────────
  家居百货              装修与设计
  机械设备              通用/农林/矿产/建筑/化工机械等14子类

[需专项资质的行业]

  行业                  所需资质
  ─────────────────────────────────────────────
  食品饮料              《食品经营许可证》/《食品生产许可证》
                        + 产品质检报告

                        (P2P、虚拟货币禁止投放)

三、禁投/高风险行业（商业化风险积分管理规则，2025.7.21起生效）
======================================================================

P0 级 - 绝对禁止（扣15分，永久封号）
  - "黑五类"产品：减肥、壮阳、丰胸、增高、私处美容
    （以虚假违规形式推广）

P1 级 - 高危险（扣9分，冻结账号）
  - 医疗健康：管制/毒性/处方药、成人用品
  - 金融：P2P、非法小额贷款、虚拟货币、私募基金
  - 母婴：一、二段奶粉及0-6个月婴儿辅食
  - 食品：槟榔及其制品
  - 非法服务：代孕、非法公文/金融服务、高仿假货

P2 级 - 限制性行业（扣3分）
  - 医疗：禁止推广具体医疗项目技术、功效原理
  - 教育：限制学科教育、戒网瘾业务
  - 生活服务：限制顺风车、殡葬、开锁
  - 电商：限制二类电商、"一元购"、网页游戏

======================================================================
四、风险积分管理规则
======================================================================

初始0分，按年度累计
  大于等于3分：笔记下线，账号预警
  大于等于6分：账号冻结14天
  大于等于12分：冻结45天
  大于等于15分：永久冻结

======================================================================
五、品牌资质要求
======================================================================

自有品牌：《商标注册证》（R标）
授权品牌：商标注册证 + 完整品牌授权链，授权期限覆盖投放周期至少6个月
商标申请中：TM标部分类目可能要求必须是R标才能投放

======================================================================
六、2025年重要新规
======================================================================

1. 医美行业准入大幅收紧（2025.12.2起执行）
   - 蒲公英医美合作笔记需先在聚光完成行业资质认证
   - 需提交：医疗机构执业许可证、医生执业资格证书、
     医疗美容主诊医师资格证书、产品医疗器械注册证

2. "低备高推"专项治理
   - 禁止利用低风险行业资质投放高风险内容
   - 违规者面临账号冻结

3. 风险积分管理规则上线

4. 蒲公英内容宣传规范更新（2025.10.27执行）
   - 禁止效果保证性承诺
   - 非医疗产品不得涉及疾病治疗/医美功效
   - 禁止绝对化用语
   - 禁止错别字、拼音替代

5. 双平台协同：聚光（拉新曝光）+ 乘风（转化成交）

6. 交易导流规范：禁止引导至第三方平台
