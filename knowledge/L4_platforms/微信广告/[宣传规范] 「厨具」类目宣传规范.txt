# 微信小店：《微信小店「厨具」类目宣传规范》
# 来源URL: https://store.weixin.qq.com/chengzhang/webdoc/wiki/7497/a7c54452af2dec04/growth_center_rule_for_store/4
# 分类: 宣传规范
# 抓取时间: 2026-05-12

============================================================

指微信小店商家/达人通过虚构活动、恶意炒作等营销手段获取人气、关注、诱导用户下单的行为。
1.2 违规判定类型
违规行为包括但不限于以下类型：
1.2.1 卖惨营销
指通过弱势群体形象炒作卖惨，博取用户同情，以此达到推广商品、诱导用户下单目的。
【常见违规行为】
（1）达人扮演贫困、衣着破烂的形象，炒作产品滞销，生计困难，无力生活，引导用户购买其产品的行为；
（2）达人扮演工厂老板，炒作工厂经营困难、产品滞销，不得不停工停产，无力给工人发工资，希望用户购买其产品，使其赚钱给工人发薪。
违规示范
扮演贫穷、假装生病和产品滞销等剧情博取用户同情
正确示范
树立正确价值观，实事求是诚信讲解商品
1.2.2 演戏炒作
指通过炒作团队冲突、家庭矛盾等违反公序良俗的剧情表演，利用猎奇行为博眼球，通过虚假摆拍、虚构人设等形式表演炒作 ，以及其他夸张不实表演等影响平台秩序的推广行为。
【常见违规行为】
（1）直播过程中，通过直播环境、服装道具等，营造虚假人设，吸引用户关注的行为；
（2）直播过程中，“商家”宣称不愿让利，达人制造冲突强迫“商家”或“自己擅自改为低价”，以此夸大商品的价格优势，带动直播间气氛，冲突期间出现叫喊、争吵、摔东西甚至肢体冲突等过激行为。
违规示范
虚构达人和商家现场砍价的场景
为了砍价，演戏打架
正确示范
实事求是诚信讲解商品价格福利
1.2.3 虚假福利
指通过虚假承诺福利、虚假抽奖、虚假赠品等方式吸引人气、吸引用户关注但最终未兑现的行为。
【常见违规行为】
（1）口播信息或画面贴图误导用户，比如：使用低价贴牌诱导用户停留直播间，但实际售卖价格远超于贴牌价格。
（2）虚假承诺福利，比如：宣传仅补贴，拍完恢复原价，营造实际商品售价本身即如此。
违规示范
贴牌价格误导用户，非实际售价
虚假宣传仅补贴，实际商品售价一直如此
正确示范
标注真实的福利信息和活动
二、拒绝虚假宣传
2.1 虚假宣传的定义
指微信小店商家/达人明示或暗示与实际内容不符的虚假信息，欺骗或误导用户，使用户对商品或服务产生误解的行为。
2.2 违规判定类型
违规行为包括但不限于以下类型：
2.2.1 商品宣传信息与实际不符
指商品的关键信息与实际情况不一致，如规格、品牌、物流方式、运费、价格、售后服务等。
【常见违规行为】
（1）宣传商品数与实际不一致；
（2）宣传商品价格与橱窗售卖商品价格不一致；
（3）宣传品牌与橱窗商品品牌不一致。
违规示范
宣传商品数与实际不一致
正确示范
诚信地介绍商品品牌参数和价格
（4）宣传信息与实际售卖商品不符。
违规示范
宣传售卖商品与实际商品不一致
正确示范
帮助消费者购买到真实正确的产品
2.2.2 虚假营销活动吸引人气
指直播内容出现活动噱头、售卖倒计时如仅限今天、只卖一天等营造虚假折扣、福利、紧张的营销氛围来吸引人气、诱导下单的情形。
【常见违规行为】
（1）利用限时活动噱头，营造紧张氛围诱导用户下单。
三、禁止发布低质量内容
3.1 低质量直播内容的定义
指微信小店商家/达人发布低质量的直播、短视频内容，影响用户观看、互动体验的行为。
3.2 违规判定类型
违规行为包括但不限于以下类型：
3.2.1 无露脸直播
指直播间无真人露脸直播，或仅开播时露脸直播过程中长时间不露脸，影响视频号橱窗用户观看、互动体验的行为。
【常见违规行为】
（1）利用镜子反向露脸，但未露出完整清晰的人脸；
（2）直播中无真人露脸或未正对镜头进行直播。
违规示范
仅展示商品无达人露脸直播
不正对镜头，未完整露出人脸五官
正确示范
真诚地介绍商品露出清晰的五官正脸
3.2.2 道具代播
以低成本道具代替直播主体的行为。
【常见违规行为】
（1）利用转盘等物品代替真人演示商品外观，无真人讲解；
（2）利用打印纸张、收集展示商品图片等物品代替商品实物。
违规示范
以物品代替真人
正确示范
有配合产品对应的氛围环境实景直播，展示真实商品，达人实时讲解
3.2.3 无实景直播
使用绿幕或贴图冒充实景实物拍摄的行为。
【常见违规行为】
（1）直播画面使用绿幕手段将图片或视频作为大面积背景代替真实环境或实物；
（2）利用贴图代替商品实物或混淆用户视觉、挂播、遮挡、贴图过大、展示信息不合规、蹭热度等影响用户观看、互动体验的行为。
违规示范
贴图代替实物商品
正确示范
正确使用绿幕和贴图，达人位于画面中央露出正脸，真诚地展示商品实物
四、禁止出售/推广假冒商品
4.1 出售/推广假冒商品的定义
指微信小店商家/达人在直播中出售/推广假冒、盗版商品，或售卖未取得品牌方授权商品的行为。
4.2 违规判定类型
违规行为包括但不限于以下类型：
4.2.1 推广假冒商品
【常见违规行为】
（1） 推广未取得相应品牌方授权的商品；
（2） 利用有品牌授权的商品链接，售卖其他商品；
（3） 商品外观、商品商标和知名品牌高度相似，使用户产生了混淆误认。
五、禁止使用不规范名字或头像
5.1 名字或头像不规范的定义
指微信小店商家/达人未经授权使用他人姓名、形象、品牌，侵害他人姓名权、名誉权、肖像权、隐私权、知识产权、商业秘密以及其他合法权利；或通过混淆行为造成视频号橱窗用户对其经营资质产生错误认识。
5.2 违规判定类型
违规行为包括但不限于以下类型：
5.2.1 名字或头像不规范
【常见违规行为】
（1）在名字和头像中无资质使用含有“官方”、“授权”、“专卖”及其他带有类似含义的内容。
（2）未经授权在名字和头像中使用知名卖场/地点品牌。
（3）未经授权在名字和头像中使用知名商品品牌。
（4）未经授权在名字和头像中使用名人头像/形象/姓名及其变种。
六、违规处罚
针对违规行为，腾讯将对发布以上违规行为的商家/达人的直播间不限于采取限流、限制直播、封禁橱窗、限制交易等处置措施。
