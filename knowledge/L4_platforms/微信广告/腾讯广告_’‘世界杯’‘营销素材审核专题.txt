# ’‘世界杯’‘营销素材审核专题
来源: 腾讯广告帮助中心
URL: https://support.e.qq.com/detail?cid=4856&pid=7632

---

’‘世界杯’‘营销素材审核专题投放营销涉及【世界杯】需要提供相应授权
一、 基础规则


在未经授权的情况下，客户生产、销售的商品、及营销营销活动中不得使用世界杯相关元素




元素示例

















X 未授权使用世界杯相关元素
X 未授权使用世界杯相关元素




未经授权的情况下，任何主体不得使用世界杯元素作为商业名称、企业名称、公众号名称等




元素示例





















X 未授权名称使用世界杯元素
X 未授权头像/图片使用世界杯元素





未经授权的情况下，营销落地页不得使用世界杯相关元素及名义作为网站域名




元素示例





















X 未授权营销落地页使用世界杯域名
X 未授权营销落地页使用世界杯域名





未经授权不得使用参赛运动员、教练员、训练员或官员的肖像、姓名、图片或运动成绩作为营销元素出现




元素示例





















X 未授权使用参赛运动员肖像/名义
X 未授权使用参赛运动员肖像/名义





营销中不支持以世界杯门票作为奖品进行抽奖、促销等任何形式的促销、竞赛或抽奖奖品




元素示例





















X 世界杯门票促销奖品
X 世界杯门票抽奖奖品





未经授权不得擅自宣传可直播转播世界杯赛事，需有国际足球联合会的授权




元素示例





















X  未授权直播世界杯赛事
X  未授权直播世界杯赛事





营销中不支持变形或模仿世界杯相关元素，需要客户提供明确的官方授权文件




元素示例





















X  变形/模仿世界杯元素
X  变形/模仿世界杯吉祥物





需国际足球联合会官方授权可使用的描述：为世界杯加油




元素示例
















X  未授权使用“为世界杯加油”描述
X  未授权使用“为世界杯加油”描述





FIFA权益人享有以营销或推广目的使用FIFA官方标志的排他性权利，除了官方授权的FIFA权益人，其他企业均不可以合作伙伴或赞助商的身份进行营销宣传




元素示例





















X  未授权使用赞助商身份进行宣传
X  1. 未授权使用世界杯相关元素    X  2. 未经授权不得售卖官方纪念币





任何与FIFA、比赛、仪式、主题曲等音视频、影视材料、文字材料等，未经FIFA授权不可使用




元素示例





















X  未授权使用售卖世界杯影视资料
X  未授权使用售卖世界杯影视资料





其他使用世界杯元素进行隐性营销的，即故意或无意地与世界杯创造虚假的、引人误解的商业联系，误导公众，使公众认为其与世界杯、国际足球联合会 具有商业合作关系或与世界杯标志权利人之间有赞助或者其他支持关系的情况，均需要核实是否有授权或者合作关系




元素示例





















X  使用世界杯元素进行隐性营销
X  使用世界杯元素进行隐性营销




声明：本文中的素材仅为说明审核要点之目的，请勿做其他用途
