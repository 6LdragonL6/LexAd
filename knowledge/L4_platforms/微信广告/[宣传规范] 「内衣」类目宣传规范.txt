# 微信小店：《微信小店「内衣」类目宣传规范》
# 来源URL: https://store.weixin.qq.com/chengzhang/webdoc/wiki/768/aabb598cf9b050aa/growth_center_rule_for_store/4
# 分类: 宣传规范
# 抓取时间: 2026-05-12

============================================================

微信小店商家/达人推广内衣类目商品时，除遵守国家法律法规要求及
《视频号橱窗带货规则总则》
《微信视频号运营规范》
《微信视频号直播功能使用协议》
《微信视频号禁止发布的营销信息列表》
《视频号橱窗开放类目公告》
《微信小店「一级类目 - 服饰内衣、箱包皮具、鞋靴」类目管理规则》
等平台规则外还需遵守下述规范：
一、禁止发布违禁信息
1.1 发布违禁信息定义
指微信小店商家/达人在直播中发布的商品信息为国家法律、行政法规、地方性法规、部门规章等规范性文件所禁止发布，或为不适合网络平台交易，或为特许经营但无此资质的情形。
1.2 违规判定类型
违规行为包括但不限于以下类型：
【常见违规行为】
违规示范
售卖禁售商品（丁字裤）
售卖禁售商品（透明情趣内衣）
正确示范
推广有资质合规内裤，无不良引导
二、禁止使用不正当营销手段
2.1 不正当营销定义
指微信小店商家/达人通过虚构活动、恶意炒作等营销手段获取人气、关注、诱导用户下单的行为。
2.2 违规判定类型
违规行为包括但不限于以下类型：
【常见违规行为】
（1）真人展示情趣内衣、成人肚兜。
（2）真人展示男生三角内裤，敏感部位凸起痕迹明显。
违规示范
真人展示男士内裤，行为具有性暗示
刻意展示敏感部位
正确示范
手持展示(不允许真人穿)
用假人模特展示
三、禁止发布超范围信息
3.1 发布超范围信息定义
指视频号橱窗商家/达人通过视频号橱窗出售或推广的商品超出平台商品开放类目范围的情形。
3.2 违规判定类型
违规行为包括但不限于以下类型：
注：
（1）售卖商品带有品牌标识的，需在微信小店后台上传对应品牌授权资料。详见：
微信小店「品牌关联类目」流程指引
（2）服饰内衣部分定向准入类目仅支持经营主体为企业。详见：《微信小店「一级类目 - 服饰内衣」定向准入标准》。
【常见违规行为】
违规示范
未经平台审核许可推广商品（定向准入类目需要有资质才能售卖）
正确示范
推广通过平台入驻审查的类目商品
四、禁止使用虚假宣传手段
4.1 发布虚假宣传信息定义
指微信小店商家/达人明示或暗示与实际内容不符的虚假信息，欺骗或误导用户，使用户对商品或服务产生误解的行为。
4.2 违规判定类型
违规行为包括但不限于以下类型：
4.2.1 违规使用广告词
指微信小店商家/达人违规使用广告词、虚假限时文案，欺骗、误导、激发消费者抢购心理的行为。
【常见违规行为】
（1）使用绝对化用语，比如：
全网最低价、史上最低价、最流行、最受欢迎、最舒适等；
（2）使用虚假限时文案，比如：
限时、抢疯了、倒计时、秒杀、卖疯了、再不抢就没了、过了今天不再有、不会再便宜了、错过就没机会了、万人疯抢等。
违规示范
使用绝对化用语 如 「秒杀」
使用虚假限时文案 如「限时」
正确示范
针对目标受众群体，设计合适的宣传文案
宣传「限时」明确时间范围
4.2.2 商品信息与实际不符
指商品的关键信息与实际情况不一致，如材质、品牌、物流方式、运费、价格、售后服务等。
【常见违规行为】
（1）主播宣传内裤为桑蚕丝（真丝）材质，但质检报告显示为纯棉。
（2）主播在直播间口播：100%桑蚕丝（真丝）、对应商品无质检报告。
（3）宣传“运费险”服务，但相应的商品实际未提供对应的服务保障。
违规示范
直播背景板/口播宣传赠送“运费险”，但相应的商品无“运费险”服务
正确示范
直播间宣传赠送“运费险”，且商品有“运费险”服务
4.2.3 虚构或夸大其他商品的功效
指微信小店商家/达人对商品/服务进行营销的过程中，通过明示或暗示对商品的使用效果、功效进行没有依据、虚假或夸大的描述，使用户对商品/服务的功效产生错误认识的行为。
【常见违规行为】
（1）宣传商品含“抗菌、抑菌”等功效，商品详情未展示相关检验项目的合格质检报告。
正确示范
宣传商品效果的，展示对应质检报告
五、违规处罚
针对违规行为，腾讯将对发布以上违规行为的商家/达人的直播间不限于采取限流、限制直播、封禁橱窗、限制交易等处置措施。
